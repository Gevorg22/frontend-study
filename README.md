# Q&A Hub - Шпаргалка для фронтенд разработчиков

Красивый и функциональный сайт-шпаргалка с вопросами и ответами по фронтенд разработке.

## Возможности

- **Левое меню** с категориями (JavaScript, React, TypeScript, CSS, HTTP, Git и др.)
- **Список вопросов** в каждой категории
- **Детальная страница** с полным ответом на вопрос
- **Быстрый поиск** (Ctrl+K или Cmd+K)
- **Навигация** между вопросами (предыдущий/следующий)
- **Трекер обучения** отмечайте изученные вопросы
- **Адаптивный дизайн** для мобильных устройств
- **Красивый UI** с градиентом и плавными переходами
- **Развернуто на GitHub Pages** доступно всем

## Структура проекта

```
src/
├── components/
│   ├── Home/                     # Главная страница с категориями
│   ├── Sidebar/                  # Левое меню с категориями
│   ├── CategoryPage/             # Страница категории со списком вопросов
│   ├── QuestionDetail/           # Страница вопроса и ответа
│   ├── QuestionItem/             # Карточка вопроса в списке
│   ├── CategoryCard/             # Карточка категории на главной
│   ├── Search/                   # Модал поиска
│   └── MarkdownRenderer.tsx      # Обработчик markdown контента
├── context.tsx                   # React Context для данных
├── types.ts                      # TypeScript типы
├── App.tsx                       # Главное приложение
└── main.tsx                      # Entry point

questions-data.json              # Данные с 1,283 вопросами и ответами
```

## Установка и запуск

### Требования

- Node.js 16+
- npm или yarn

### Установка

```powershell
# Перейти в папку проекта
cd "path/to/frontend-study"

# Установить зависимости
npm install
```

### Запуск в режиме разработки

```powershell
npm run dev
```

Сайт откроется на `http://localhost:5173/`

### Сборка для продакшена

```powershell
npm run build
```

## Горячие клавиши

- **Ctrl+K** (или **Cmd+K** на Mac) - открыть поиск
- **Esc** - закрыть поиск или меню

## Добавление новых вопросов

Вопросы хранятся в файле `questions-data.json`. Добавьте новый объект в массив категории:

```json
{
  "id": 1284,
  "categoryId": "javascript-basics",
  "title": "Ваш вопрос здесь?",
  "answer": "Полный ответ на вопрос..."
}
```

### Доступные категории (categoryId):

- `javascript-basics` - JavaScript Основы (301 вопрос)
- `react` - React (227 вопросов)
- `asynchronous` - Асинхронность (107 вопросов)
- `typescript` - TypeScript (91 вопрос)
- `http` - HTTP & API (71 вопрос)
- `architecture` - Архитектура (51 вопрос)
- `css` - CSS (49 вопросов)
- `performance` - Производительность (65 вопросов)
- `git` - Git (1 вопрос)
